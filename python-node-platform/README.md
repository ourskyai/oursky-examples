# Python Node Platform - System Status

Uses the OurSky Node Platform Python SDK to fetch and display the system status of a node.

## Setup

1. Install dependencies:

```bash
pip install -r requirements.txt
```

2. Copy `.env.example` to `.env` and set your API token (required), Lineage ID (required), and optionally the API base url.

```bash
cp .env.example .env
```

3. Run the example:

```bash
python main.py
```
